# Customer Churn Analysis Prediction

## Project Overview
This project analyzes customer churn data and predicts whether a customer is likely to leave a telecom service.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Dataset
Telco Customer Churn Dataset

## Features
- Data Cleaning
- Data Preprocessing
- Churn Prediction
- Model Evaluation
